# performance-diagram

## Description
A library to (easily) plot Performance Diagrams.
Supports both CSI and F1 score lines.

`Roebber, P. J. (2009). Visualizing multiple measures of forecast quality. Weather and Forecasting, 24(2), 601-608.`
